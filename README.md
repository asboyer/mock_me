## Linux
#### Requirements: 
* python3

#### Steps:
```bash
pip install SpeechRecognition
sudo apt-get install portaudio19-dev python3-pyaudio
pip install PyAudio
```

## Mac OS
#### Requirements: 
* homebrew 
* python3

#### Steps:
```bash
pip install SpeechRecognition
brew install portaudio
python3 -m pip install pyaudio
```

## Windows:
#### Requirements: 
* python3

#### Steps:
* pip install SpeechRecognition
* pip install pipwin
* pipwin install pyaudio
